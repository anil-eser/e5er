01The 3 Pillars of SEO

Semantic SEO

Optimizing content so search engines understand what your page is about. Includes keyword research, content structure, topic relevance, and proper HTML tag usage.

Technical SEO

Everything that makes your site crawlable and indexable — site speed, mobile-friendliness, structured data, sitemaps, robots.txt, redirects, Core Web Vitals.

Netlinking (Off-Page SEO)

Building authority through external links. Google reads backlinks as votes of confidence. Quality matters far more than quantity.

02Key Google Algorithm Updates
UpdatePurpose
PandaPenalizes thin, low-quality, or duplicate content
PenguinPenalizes abusive netlinking — link spam, link farms, paid links
PigeonImproves local search results, connects local SEO to core signals
BERTBetter understanding of natural language and search intent
RankBrainAI that interprets ambiguous queries by mapping them to known ones
RankBrain is not a one-time update — it is a permanent AI component of Google's ranking algorithm, always active.
03Googlebot — What It Can and Cannot Do

CAN

  • Copy and read HTML content
  • Navigate via href links
  • Index web pages
  • Read robots.txt and sitemaps
  • Execute some JavaScript (unreliably)

CANNOT

  • Generate JavaScript code
  • Log in to authenticated pages
  • Fill out forms
  • Access content behind paywalls reliably
04Crawl Budget

Crawl budget = the number of pages Googlebot crawls on your site in a given time. Google factors in:

  • Site size — larger sites need more budget
  • PageRank — higher authority pages crawled more frequently
  • Server capacity — slow servers reduce crawl frequency
Content length is NOT a crawl budget factor. Word count does not affect how often Googlebot visits your pages.
05robots.txt

A text file at your domain root that tells crawlers what they can and cannot access.

Two main purposes

  • Block the crawl of certain site directories
  • Indicate the location of your sitemap.xml

Key distinction

robots.txt blocks crawling — not indexing. To block indexing, use the noindex meta tag.

Critical directive

User-agent: *
Disallow: /

This blocks all bots from crawling the entire site.

06Sitemaps
Maximum 50,000 URLs per sitemap file and 50MB uncompressed.
TagPurpose
<url>Container for each page entry
<loc>The URL of the page
<lastmod>Date the page was last modified
<changefreq>How often the page changes
<priority>Priority score 0.0–1.0
<xhtml:link>Hreflang alternatives for multilingual sites
07HTML Tags — Semantic Importance
  • Title — Most important on-page SEO element. Shown in SERPs. Max ~60 chars.
  • H1 — Main page heading. One per page. Contains primary keyword.
  • Meta Description — NOT a ranking factor but impacts CTR. ~155 chars.
  • Alt tag — Describes image content. Used by Google and screen readers.
Meta description keywords are not directly considered by Google's algorithm.
08Black Hat vs White Hat SEO

Black Hat — Avoid

  • Keyword stuffing
  • Cloaking
  • Link spamming
  • Link farms / PBNs
  • Hidden text
  • Doorway pages
  • Sneaky redirects

White Hat — Safe

  • Multiple tagging — normal practice, not black hat
09Core Web Vitals
MetricMeasuresGood
LCP — Largest Contentful PaintLoading speed< 2.5s
FID — First Input DelayInteractivity< 100ms
CLS — Cumulative Layout ShiftVisual stability< 0.1
SRFC does not exist — not a Core Web Vital. The three metrics are LCP, FID, and CLS.
10HTTP Redirects — 301 vs 302
CodeTypeSEO Impact
301Permanent redirectPasses link juice (~99%) to new URL
302Temporary redirectDoes NOT reliably pass link juice
Always use 301 when permanently moving a page.
11hreflang — Multilingual SEO 
<link rel="alternate" hreflang="en-gb" href="..." />
<link rel="alternate" hreflang="fr" href="..." />
<link rel="alternate" hreflang="x-default" href="..." />
  • x-default — fallback for users who don't match any region
  • Essential for multilingual and multiregional sites
12Structured Data & Schema.org

What it CAN do

  • Improve traffic quality
  • Improve CTR via rich snippets
  • Improve on-site user experience

What it CANNOT do

  • Directly improve SEO rankings
  • Guarantee a rich snippet appearance

Schema.org creators

  • Google Microsoft (Bing) Yahoo Yandex
  • Apple — did NOT participate
13Google Search Console vs Analytics
ToolPurpose
Search ConsoleSEO — impressions, clicks, positions, indexing, Core Web Vitals, Sitemaps
Google AnalyticsTraffic — sessions, bounce rate, conversions, traffic acquisition
Traffic acquisition belongs to Google Analytics, not Search Console.
